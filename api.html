// endpoints

POST /v1/scan Проверить URL на фишинг
Request 
curl -X POST https://api.phishguard.io/v1/scan \
  -H "X-PhishGuard-Key: pg_live_sk_..." \
  -H "Content-Type: application/json" \
  -d '{"url": "https://paypa1.com/login"}'

{

import { PhishGuard } from '@phishguard/sdk';

const pg = new PhishGuard('pg_live_sk_...');

const result = await pg.scan('https://paypa1.com/login');

import "github.com/phishguard/go-sdk"

// rate limits & errors

HTTP код Значение Действие
200 OK — успешный ответ Используйте данные
401 Unauthorized — неверный ключ Проверьте X-PhishGuard-Key
422 Invalid URL — некорректный формат Проверьте тело запроса
429 Rate limit exceeded Retry-After в заголовке
503 Service unavailable Экспоненциальный backoff
